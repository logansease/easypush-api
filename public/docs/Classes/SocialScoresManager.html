prepareForFbUser

You can call this method after the user signs in to facebook to have the server prepare for the user by creating them and loading their friend list. By calling this preemtively, you avoid possible delays on the users' first score save. This is optional. Runs in separate thread.

- (void)prepareForFbUser

Discussion

You can call this method after the user signs in to facebook to have the server prepare for the user by creating them and loading their friend list. By calling this preemtively, you avoid possible delays on the users' first score save. This is optional. Runs in separate thread.
